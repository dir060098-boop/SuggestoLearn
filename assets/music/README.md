# Музыкальные треки

Поместите сюда 4 MP3-файла:

| Файл | Концерт | BPM | Рекомендация |
|---|---|---|---|
| `active_vivaldi.mp3` | Активный | 120–140 | Вивальди — Четыре сезона (Весна) |
| `active_telemann.mp3` | Активный | 120–140 | Телеман — любой оркестровый концерт |
| `passive_bach.mp3` | Пассивный | 55–65 | Бах — Ария из сюиты №3 (Air on G String) |
| `passive_chopin.mp3` | Пассивный | 55–65 | Шопен — Ноктюрн op.9 №2 |

## Где найти

- [Musopen.org](https://musopen.org) — бесплатные записи в общественном достоянии
- [Archive.org](https://archive.org) — исторические записи

## Временные треки

В репозитории есть сгенерированные заглушки (синусоидальные дроны с нужным BPM).
Они работают для тестирования, но замените на настоящие барочные записи для полного эффекта метода.
